# Project Brief: SAT Coach

## Project Name
SAT Coach - AI-Powered SAT Preparation Platform

## Vision
An intelligent tutoring system that helps high school students prepare for the SAT through adaptive AI coaching, spaced repetition learning, and personalized difficulty adjustment.

## Core Requirements

### 1. Three-Service Architecture
- **Frontend Application**: React with TypeScript
- **DB Backend**: Node.js/Express API with MongoDB
- **AI Backend**: OpenAI GPT-4-mini integration service

### 2. User Interface
- Split-screen layout:
  - Left: SAT question display
  - Right: ChatGPT-style AI tutor chat
- Questions generated by AI, categorized by subject and difficulty
- Free navigation between questions
- Timer for practice sessions

### 3. Intelligent Learning System
- **Spaced Repetition**: Track topics, dates, and success/failure levels
- **Adaptive Difficulty**: AI adjusts explanation complexity based on student performance
- **Progress Tracking**: Comprehensive analytics and performance metrics
- **Question Management**: Save generated questions to database for reuse

### 4. User Management
- Full authentication (login/signup)
- Individual student profiles
- Progress history and analytics

### 5. Enhanced Features
- Bookmark questions
- Mark questions for review
- Rate AI explanations
- Session timer
- Performance analytics dashboard

### 6. Deployment
- Vercel hosting
- CI/CD pipeline setup
- MongoDB Atlas for database

## Success Criteria
1. Students can sign up and log in securely
2. AI generates appropriate SAT questions by subject and difficulty
3. AI tutor provides contextual help for each question
4. System tracks and adapts to student learning patterns
5. Spaced repetition algorithm optimally schedules review sessions
6. Questions are saved and reused efficiently
7. Deployment is automated through CI/CD

## Technical Constraints
- Must use React with TypeScript
- Must use MongoDB for data storage
- Must integrate OpenAI GPT-4-mini
- Must deploy on Vercel
